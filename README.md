About the website:


This project is a clean, modern, and responsive digital resume template, designed to mirror the structure and layout of the provided pfp2.png document.


Overview

This web-based resume was built to provide a professional online presence. The layout utilizes a two-column design:

  Sidebar: Contains contact details and educational history.

  Main Content: Displays the professional summary, work experience, and reference information.


Technologies Used

  HTML5: Sematic structure for accessibility and clarity.

  CSS: Styling to ensure a modern, clean, and consistent layout across devices.

  Javascript: Basic interactivity for enhanced user experience.
